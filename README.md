
# Google Analytics Customer Revenue Prediction

## ?? Project Overview
This project aims to predict future revenue per customer using transaction history from the Google Merchandise Store dataset. The goal is to help businesses identify high-value customers and optimize marketing strategies.

## ?? Technologies Used
- Python
- Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
- Cloud platforms (AWS, Azure, GCP)
- MLOps tools (MLflow, DVC, CI/CD)
